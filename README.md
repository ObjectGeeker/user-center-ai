# 用户中心(AI赋能)

## 项目介绍
一站式用户管理平台，可进行项目级用户管理，权限分配，并引入AI框架，让用户可以通过AI对话的方式直接进行用户及权限操作

## 需求分析
1. 由于是用户管理平台，所以只能由管理员及管理员以上的角色才可以进行登录
2. 管理员账号由超级管理员进行管理
3. 超级管理员账号在系统运行前自动进行创建
4. 管理员可以对用户信息修改
5. 管理员可以对用户账号进行封号
6. 管理员可以对用户账号进行踢人下线
7. 允许多台设备同时登录一个账号
8. 管理员可以使用AI对话操作上述功能

## 技术选型

### 后端
1. SpringBoot 3.5
2. Sa-Token (Simple模式)
3. MySQL
4. Redis
5. Spring AI + OpenAI

### 前端
1. React + Vite打包
2. Ant Design UI

## 项目目标
1. 学习Sa-Token的用法
2. 学习SpringAI框架整合RAG MCP
3. 学习手搓一个简易ReactAgent
